# Проект для конкурса СберИндекс

## Название: Качество жизни за высокую цену

### Описание:
Был выбран уникальный регион Российской Федерации, мой родной остров Сахалин. С помощью данных получилось проанализировать принципы расселения граждан и ответить на вопрос формирования высоких зарплат. На основе данных была создана история и представлена в виде сайта


### Структура проекта:
analysis: содержит код обработки и исследования данных
app: код для создания веб-сайта на библиотеке streamlit
data: исходные файлы с данными (были использованы файлы предоставленные организаторами конкурса, а также найденны данные на Росстате и OpenStreetMap)
final_data: обработанные данные для построения визуализаций
images: вспомогательные изображения и обработанные графики


### Инструкция для локального воспроизведения:
1. Установить зависимости из requirements.txt `pip install -r requirements.txt`
2. pfgecnbnm приложение streamlit `streamlit run app/app.py`
